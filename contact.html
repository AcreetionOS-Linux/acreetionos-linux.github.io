Bug Reports & Feature Requests

We now track bugs and feature requests on GitHub. Please open issues in the appropriate repository within our organization at https://github.com/AcreetionOS-Linux/. If you're unsure which repository fits your report, use the organisation issues interface or the Create a GitHub Issue button below and select the correct repository when prompted.

How to file

  • Check the appropriate repository (for example: `website`, `iso`, `installer`).
  • Search existing issues to avoid duplicates.
  • Open a new issue and include steps to reproduce, expected vs actual behavior, and logs if possible.
Create a GitHub Issue Let's Discuss the issue on Discord

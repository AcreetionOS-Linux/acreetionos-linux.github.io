# AcreetionOS

AcreetionOS is a community-focused, Arch-based Linux distribution designed to be approachable for new users while staying powerful and flexible for experienced and new users.

**Quick facts**

- Base: Arch Linux (rolling releases, AUR access)
- Filesystem: EXT4 (default)
- Audio: PipeWire
- Boot approach: Hybrid boot support for live media and installed systems (systemd-boot/syslinux for live, GRUB for installed systems)

## Features

- Arch-based core with access to the Arch ecosystem and AUR
- User-friendly out-of-the-box experience with tested defaults
- Stable filesystem configuration using EXT4 by default
- Modern audio stack powered by PipeWire for low latency and broad device support
- Hybrid boot strategy for broad hardware compatibility

## Philosophy

We are an open-source project focused on learning, collaboration, and providing a sensible, usable desktop experience. Contributions are welcome from people of all skill levels — we value mentorship and documentation as much as code.
